# Chapter Done

Wrap up a chapter: run tests, commit progress, update PROGRESS.md journal.

1. Run `./gradlew test` and confirm all tests pass
2. Ask the user for a brief summary of what they learned
3. Update PROGRESS.md with a new session entry including:
   - Date, chapter number, what was completed
   - Key learnings and any "aha" moments
   - Next session ideas
4. Stage and commit with a chapter-appropriate message (e.g., "Complete Chapter 5 — variables and assignment")
5. Push to main
